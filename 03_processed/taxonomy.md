# Taxonomy of approaches

Structured classification of whole-body control methods encountered in the literature.

## Axes
1. Control formulation (task-priority / QP / MPC / RL / hybrid)
2. Contact modeling (rigid / soft / LCP / relaxed)
3. Actuation model (torque / position / impedance / SEA)
4. Simulator coupling
5. Validation scope (simulation only / hardware)
