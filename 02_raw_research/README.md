# Raw research

Raw, immutable outputs. Do not edit files here — process them in `03_processed/`.

Subfolders:
- `perplexity_reports/`, `gemini_reports/`, `claude_reports/` — LLM outputs (one file per run, date-prefixed).
- `pdf_papers/` — peer-reviewed PDFs.
- `videos_talks/` — links or transcripts of conference talks.
