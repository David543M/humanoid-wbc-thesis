# Simulation

All simulation assets and live controller source.

- `urdf_sdf_models/` — robot kinematic/dynamic descriptions.
- `scenes/` — simulator scene/world files.
- `configs/` — YAML/JSON configs for controllers, solvers, task hierarchies.
- `controllers_src/` — source of controllers currently being iterated on (frozen copies go in `09_appendices/code/controllers/`).
